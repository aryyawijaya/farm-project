## How to Run ##
### 1. Run postgresql script ###
### 2. Setup env ###
- Create .env file from .env_sample file
- Set user, password, and port postgres

### 3. Install dependecies ###
```
npm install
```
### 4. Start the server ###
```
npm start
```
